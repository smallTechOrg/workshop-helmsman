# Workshop Helmsman — Agent layer

**This product has no agent framework.** It is a regular web app (FastAPI +
Jinja2 server-rendered + vanilla JS). No LLM, no tool-use loop, no MCP.

If future phases add an AI Help Desk (a sub-agent that suggests fixes when a
participant flags they're stuck), that will live in `capabilities.md` and
`roadmap.md` — *not* here. This file exists only so the spec layout matches
the boilerplate contract.
